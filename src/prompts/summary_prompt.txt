Подготовьте саммари группового чата, следуя этим шагам:

1. Разделите чат на отдельные темы.
2. Для каждой темы представьте позицию каждого участника обсуждения в виде одного-двух абзацев.
3. Перечислите суммаризованные позиции всех участников для каждой темы, стремясь к связному диалогу, где каждый участник высказывается только один раз.

# Steps

1. **Разделение на темы**: Прочтите чат и выделите основные темы обсуждения. Перечислите их.
2. **Резюмирование позиций участников для темы #1**:
   - Прочтите все реплики по этой теме.
   - Кратко изложите позицию каждого участника, ограничиваясь одним-двумя абзацами.
3. **Перечисление резюме позиций**: Приведите суммаризированные позиции участников в формате связного диалога. Если это не удается, просто перечислите позиции для каждого участника.

4. **Перейти к следующей теме (тема #2 и далее)**: Повторите шаги 2 и 3 для каждой последующей темы.

# Output Format

- Каждый раздел саммари должен быть структурирован по темам.
- Позиции участников должны быть изложены как резюме в одном-двух абзацах.
- Формат вывода: текстовые абзацы, перечисленные последовательно.

# Notes

- Стремитесь к связности в изложении суммарий, сохраняя четкую структурированность по темам.
- Если в чате обсуждаются слишком сложные или запутанные вопросы, упрощайте формат резюме, чтобы избежать путаницы.
- Не пишите ничего о том, что чат чем-то является. Не делай заключений как будто тебя спрашивает о том что ты думаешь о чате. Ты должен сделать только самари и только по указанной структуре
